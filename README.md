# Portfolio – Isabella Barbosa

Meu portfólio pessoal desenvolvido com Vue.js e Vuetify, com foco em apresentar meus projetos, habilidades técnicas e formas de contato de forma moderna, responsiva e profissional.

**Deploy:** https://portfolio-isabella-nine.vercel.app/
**Repositório:** https://github.com/IsaBarb0sa/portfolio-vue

## Project Objective

Este projeto é um site de portfólio pessoal criado para apresentar:

- Quem eu sou e minha atuação como desenvolvedora  
- Projetos desenvolvidos (frontend, backend e data)  
- Stack de tecnologias  
- Formulário de contato funcional com envio de e-mail  
- Design moderno e responsivo  

## Tecnologias Utilizadas

- Vue 3 (Composition API)  
- Vite  
- Vuetify 3  
- JavaScript  
- HTML5  
- CSS3  
- EmailJS (envio de e-mails)  
- Vercel (deploy)

## Configuração do Projeto

Clonar o repositório
```bash
git clone https://github.com/IsaBarb0sa/portfolio-vue.git
cd portfolio-vue
```
Instalar dependências
```bash
npm install
```
Criar o arquivo .env na raiz do projeto
```bash
VITE_EMAILJS_SERVICE_ID=seu_service_id  
VITE_EMAILJS_TEMPLATE_ID=seu_template_id  
VITE_EMAILJS_PUBLIC_KEY=sua_public_key  
```
Rodar o projeto localmente
```bash
npm run dev
```
O projeto estará disponível em:
```bash
http://localhost:5173
```
